# 원무팀용 진료 가이드

PDF 한 장에 있던 진료 시간표, 검사·시술 비용, 직원·지인 금액, 연락처를 검색 가능한 반응형 웹 가이드로 정리했습니다.

## 열기

`index.html`을 더블클릭하면 인터넷 연결 없이 사용할 수 있습니다.

## 확인이 필요한 원본 표기

- 위 수면비 비고란의 `asdasd`는 원본에 그대로 적힌 값입니다.
- 한약 15일분의 직원가는 원본에 `14,000원`으로 표시되어 있으나 다른 할인율과 비교해 오기 가능성이 있습니다.
- 실제 일정과 금액은 변경될 수 있으므로 예약·결제 전 재확인이 필요합니다.
